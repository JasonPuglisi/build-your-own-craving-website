# Build Your Own Craving Website

Mock e-commerce website for Virtual Enterprises International.

## Overview

Created for the mock business Build Your Own Craving during the 2013-2014
school year at Robbinsville High School as part of the Virtual Enterprises
International program. Awarded fourth place in the Advanced Website category
at Virtual Enterprises International's International Trade Show.

Features a rich, immersive front page design with database integration for
displaying statistics based on orders placed, such as how much money has been
donated by the company (based on total order amounts).

Includes a product customizer with dynamic pricing and a near-limitless
set of combinations. Originally integrated with
[OpenCart](https://www.opencart.com/) for easy management and selling of
products created by the company.
